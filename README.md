# AI-Driven Teacher Assistant

AI-powered Teacher Assistant using Django, Gemini AI, Voice Recognition, Quiz Generation and Document Analysis.

## Features

- User Authentication
- PDF Upload
- AI Summary Generation
- AI Quiz Generation
- Quiz Scoring
- Summary PDF Download
- Dashboard Analytics
- Profile Management

## Tech Stack

- Python
- Django
- Gemini AI
- SQLite
- Bootstrap

## Installation

```bash
git clone <repository-url>
cd AI-Driven-Teacher-Assistant
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
